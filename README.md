Study Timer App
A full-stack web application that gamifies studying using the Pomodoro technique with streak tracking and a gacha-style reward system.
Try it Here: https://workout-timer-seven-beta.vercel.app/ 
Features

Pomodoro Timer: Three study modes (15/25/60 min sessions) with automatic break transitions
Google OAuth: Secure user authentication with session persistence
Gamification: Earn gold for completing study sessions, unlock chimera creatures through gacha system
Streak Tracking: Build and maintain study streaks across three difficulty levels
No Pausing: Study sessions enforce uninterrupted focus time
